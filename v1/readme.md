# Charles Xavier - Espacio SAD

No estéis tristes, SAD se refiere a Seguridad y Alta Disponibilidad. En este espacio aprenderemos a protegerer los activos de **Terraformadores de Venus** así como a saber cómo actúan los atacantes. Así estaremos preparados para desbaratar sus planes en caso de ataque.
 
Intentaré guiaros en todo momento para conseguir ese objetivo, pero no olvidéis mi poder mental. Si quisiera podría saber en todo momento lo que pensáis, así que no se os ocurra utilizar lo que aprendamos para convertiros en uno de esos atacantes.
